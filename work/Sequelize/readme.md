# Sequelize

A Node project that allows your api's to talk to SQL servers (My, Postgres, MS)

- promise based 
- you can also async and await


## Benefits

Simplifies code

Javascript not sql

Robots API

Added Security (auto handles injection)

Community

CLI tools
